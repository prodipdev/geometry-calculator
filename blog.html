1. What is DOM and why do we use it?

DOM stands for Document Object Model, and it is a programming interface for web documents.
The DOM allows JavaScript to dynamically update the content and structure of a web page, without requiring a page reload. This enables developers to create interactive and responsive web applications that can provide a better user experience.

2. How to select the elements from DOM?

There are several ways to do this e.g:

getElementById : This method get the specific HTML element with id="" attribute.

getElementsByClassName : This method get the specific HTML element with class="" attribute.

getElementsByTagName : This method get collection of HTML elements with specified tag name.

querySelector : This method get the first HTML element that matches specified CSS attribute.

querySelectorAll : This method get the collection of HTML elements that matches specified CSS attribute.

3. What is event bubble?

Event bubbling is when an event triggered on an element is passed up to its ancestor elements until it reaches the root element. It allows for efficient event handling on multiple elements, but can cause unintended triggering of events.

4. What is the callback function and why we use it?

A callback function is a function that is passed as an argument to another function and is intended to be called by that function at a later time.
This callback function used for asynchronous programming, event handling, modularity and reuse etc.
